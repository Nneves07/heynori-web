Configurar Formspree para heynori

Para configurar Formspree:

  1. Ve a formspree.io
  2. Crea una cuenta gratuita
  3. Crea un nuevo formulario
  4. Copia el endpoint (será algo como: https://formspree.io/f/xpzgwqjq)
  5. Actualiza el código en Contact.tsx con el endpoint real
